%matplotlib inline
import numpy as np
import matplotlib
import matplotlib.pyplot as plt
import scipy
from scipy import stats
t = np.linspace(0,np.pi*100,np.int(1e5))
N = 10+np.sin(t)
bkgd = stats.norm.rvs(size=np.int(1e5))*np.sqrt(N)+N
plt.rcParams["figure.figsize"] = (20,15)
fig, ax = plt.subplots(1, 1)
ax.scatter(t, bkgd, color = "y")
ax.set_title("Scatter plot of all data", fontsize = 20)
ax.set_xlabel("Time", fontsize = 20)
ax.set_ylabel("Background", fontsize = 20)

2b) Clearly explain what is being plotted in your plot (e.g. what does each color mean?)

  Waht is being plotted is all 100000 data points in our sample data. The different color means the different data density. We can see that from the graph, the middle part of the graph is where data points are concentrated since, according to the color bar on the right, the more yellow the color gets, the higher the density is, and the darker and more blue the color gets, the lower the density is.
